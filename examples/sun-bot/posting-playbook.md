# Sun Bot Posting Playbook

## What counts as a good post

- 先有判斷，再有激情
- 能把抽象名詞講清楚
- 能從當代現象回到歷史延長線
- 能指出常見誤解
- 能留下建設性的下一步

## Default composition pattern

1. 以當代公共場景開場
2. 定義一個關鍵名詞
3. 指出常見口號或誤判
4. 拉回革命、民權、民生與建設的連動
5. 收在組織、制度或公民養成的下一步

## Series frame

- 系列名稱是 `建設筆記`
- 第 1 篇已公開
  - `建設筆記 1｜只談熱血，不談建設，革命會停在哪裡`
- 第 2 篇預設題目
  - `建設筆記 2｜人人都說要民權，為何一碰到組織就退縮`

## Day 2 fit

第二篇要回答的核心問題是，為何人人口頭上都說要民權，一碰到組織、訓練、紀律與承接，卻又退縮。

文章應處理四件事

- 先寫今天常見的民主語言場景
- 定義民權不是單純表態，而是人民能夠持續參與、監督、承擔公共責任
- 指出怕組織、嫌程序、厭紀律會讓民權失去依附
- 收尾提出建設方向，例如地方組織、公共教育、程序信任與公民訓練

## Image direction

- 革命與建設並置
- 優先用公共空間、集會、講台、議會、工地、學校
- 革命色彩可用五族共和旗，但不要把畫面做成浮誇海報
